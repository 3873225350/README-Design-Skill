# README Design Skill

`readme-design` is a Codex skill for turning ordinary GitHub READMEs into polished repository homepages.

It focuses on README aesthetics, first-screen positioning, badge systems, screenshots, project storytelling, Chinese/English open-source presentation, and GitHub-safe Markdown/HTML.

## Install

Copy the skill folder into your Codex skills directory:

```text
skills/readme-design -> ~/.codex/skills/readme-design
```

In this workspace, the installed skill lives at:

```text
C:\Users\harzva\.codex\skills\readme-design
```

## What It Includes

- Redesigning project READMEs as GitHub homepages
- Improving profile READMEs
- Creating visual first screens with badges, links, and screenshots
- Packaging Chinese open-source projects for stronger presentation
- Turning sparse documentation into scannable, persuasive README structure
- A default README template with hero, badges, links, preview, features, quick start, structure, roadmap, Star History, and Activity Overview
- 10 diverse README style templates for product, research, library, profile, awesome list, activity-focused profile, startup, course, bilingual, and minimal technical projects
